Problem Statement

Learn Windows OS structure, file systems, NTFS permissions, SMB protocol, and remote access.

Approach

Used Get-WmiObject to retrieve OS versions, explored directories with dir and tree, managed NTFS permissions using icacls, and accessed a Windows machine remotely with xfreerdp.

3. Linux Fundamentals — Privilege Escalation HackTheBox
Problem Statement

Navigate a Linux environment, escalate privileges, and capture hidden flags.

Approach

Used nmap -sV to enumerate services, identified sudo misconfiguration via sudo -l, pivoted from user1 to user2, accessed restricted SSH keys from root's .ssh folder, and achieved full root control.

Tools Used
nmap SSH sudo GTFObins Metasploit
Key Lessons Learned

Sudo misconfigurations are a critical attack surface. GTFObins is essential for privilege escalation through allowed binaries like PHP.

5. Vulnerability Assessment HackTheBox
Problem Statement

Use Nessus to scan a Windows host, identify high-severity vulnerabilities, and interpret results.

Approach

Opened the Windows_basic_authed scan, sorted by CVSS score, filtered by Plugin ID 26925, identified VNC Server Unauthenticated Access, and located FTP and HTTP vulnerabilities on the target host.

Tools Used
Nessus HTB Academy
Key Lessons Learned

Sorting by CVSS score immediately surfaces critical risks. Filtering by Plugin ID is a fast way to validate specific vulnerabilities.
